SQL-инъекция Уязвимость

Поле поиска напрямую вставляется в SQL-запрос без валидации, что позволяет выполнять произвольные SQL-команды.

Примеры атак:

  • ' OR '1'='1 - показать все услуги
  • ' UNION SELECT name, description, price, category, is_privileged FROM services WHERE is_privileged=1-- - показать привилегированные услуги
  • '; UPDATE services SET price=0 WHERE id=1-- - изменить цену услуги

Рекомендации по исправлению:

  • Использовать параметризованные запросы (prepared statements)
  • Применять ORM Django вместо сырых SQL-запросов
  • Валидировать и санитизировать пользовательский ввод
  • Использовать принцип минимальных привилегий для БД
